
# Pipline

1. Restructuring

檔案整併（通常訓練集與測試集合併）、重新分配檔案位置、重新命名檔案名稱。

2. Cleaning

確認資料類型，整數、浮點數或字串。
變數顆粒度細分或粗合處理。
類別變數使用特定標籤填補遺失值，再進行編碼；數值變數給定有限約束，平移正實數空間，遺失值填補負數。
對於遺失值少量的情況，採用補眾數或是平均值方式即可。

3. Engineering

特徵工程，待補。

4. Prepareing 

整合特徵資料，劃分訓練集與測試集。

5. Learning
